Important terms meanings:

Scale Factor : If the face in the image(frame) to big then the scale factor decides how much the computer will shrink the image to find the face.

Min Neighbours : Minimum number of overlapping rectangles that must be found in the same area to confidently label it as a detected face. Higher value means less false positives.

Min Size : Minimum pixel grid from which the face size must be larger. Face size smaller than them will not be detected.

Max Size : Maximum pixel grid from which the face size must be smaller. Face size larger than them will not be detected.

cv2.equalizeHist() : Improves the contrast of the image and helps the computer to detect faces better.
